# Code of Conduct

## Our Commitment

Fanek is a welcoming project. Everyone who participates -- by filing issues, submitting pull requests, writing documentation, or joining discussions -- is expected to treat others with respect and good faith.

## Expected Behavior

- Be direct and constructive in code review and issue discussions.
- Assume good intent when reading others' messages.
- Accept that maintainers may decline contributions that do not fit the project's scope.
- Give credit where it is due.

## Unacceptable Behavior

- Personal attacks, insults, or harassment of any kind.
- Discriminatory language or imagery based on identity, background, or beliefs.
- Sharing others' private information without consent.
- Deliberate intimidation or sustained disruption of discussions.

## Scope

This code of conduct applies to all project spaces: GitHub issues, pull requests, discussions, and any other channel where project members interact on behalf of the project.

## Enforcement

Maintainers may remove, edit, or reject comments, commits, code, and issues that violate this code of conduct. Repeat or severe violations may result in a temporary or permanent ban from the project.

## Reporting

To report a violation, open a private discussion or contact the maintainer directly via GitHub. All reports will be reviewed and handled confidentially.
